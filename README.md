# expo-firebase-redux-starter

## Starter project for using:
- React Native
- Expo
- Firebase
- Redux

## Setup:

- Create a `constants/ApiKeys.js` file and put your actual firebase config values:
```
export default {
  FirebaseConfig: {
    apiKey:      "",
    authDomain:  "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: ""
  }
}
```
- Run `npm install`
